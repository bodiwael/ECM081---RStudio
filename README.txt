📋 PROJECT OVERVIEW
================================================================================

RESEARCH QUESTION:
"Does the traditional Phillips Curve relationship between inflation and 
unemployment hold in the US economy over the period 1990-2025?"

MODEL SPECIFICATION:
Inflation_t = β₀ + β₁(Unemployment_t) + β₂(GDP_Growth_t) + 
              β₃(Interest_Rate_t) + β₄(Oil_Prices_t) + ε_t

EXPECTED RELATIONSHIPS:
- Unemployment → Inflation: NEGATIVE (traditional Phillips Curve)
- GDP Growth → Inflation: POSITIVE (demand-pull inflation)
- Interest Rate → Inflation: NEGATIVE (monetary policy effect)
- Oil Prices → Inflation: POSITIVE (cost-push inflation)

DATA SOURCE:
Federal Reserve Economic Data (FRED) - https://fred.stlouisfed.org/
- All US macroeconomic data
- Quarterly frequency
- Period: 1990 Q1 - 2024 Q4

🔧 TROUBLESHOOTING
================================================================================

PROBLEM: Packages won't install
SOLUTION: Update R to latest version, try install.packages() one at a time

PROBLEM: Can't find data files
SOLUTION: Check working directory with getwd(), ensure CSVs in same folder

PROBLEM: Error reading CSV files
SOLUTION: Verify column names match (DATE, CPIAUCSL, etc.)

PROBLEM: Too many missing values
SOLUTION: Check date ranges, ensure data covers 1990-2024

PROBLEM: Results don't make sense
SOLUTION: Check data for errors, review descriptive statistics

PROBLEM: Script runs slow
SOLUTION: Normal for large datasets, be patient (2-5 minutes typical)

PROBLEM: Regression table formatting
SOLUTION: Open .html file, copy to Word, clean up formatting

For more help: Check QUICK_START_GUIDE.txt section "Common Issues & Solutions"
